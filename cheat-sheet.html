Rule of thumb: a Q4-quantized model needs roughly (billions of params) × 0.6–0.8 GB of VRAM. Leave ~2GB headroom for context.

3 · The one routing rule that saves you the most

Premium tokens for reasoning. Free local models for grunt work.
Route every call by difficulty: send the hard 20% (planning, judgment, final synthesis) to your best model, and the easy 80% (drafting, extracting, reformatting, classifying) to a free 3–4B local model. Most "$200/month AI bills" are 80% grunt work paying frontier prices.

4 · Be running in 10 minutes

# 1. Install Ollama (Win/Mac/Linux): https://ollama.com
# 2. Pull the lean starter stack:
ollama pull llama3.2:3b        # fast grunt-work model
ollama pull qwen3:14b          # reasoning workhorse
ollama pull nomic-embed-text   # embeddings for memory/RAG

# 3. Test it (chat in your terminal):
ollama run qwen3:14b "Summarize why local models beat API bills for grunt work."

# 4. It's an OpenAI-compatible server on http://localhost:11434
#    Point any tool/SDK at that base URL — no API key, $0/call.

That's the whole foundation: a router, a grunt model, a reasoning model, and an embedder. Everything else is built on top.
